Ressources

Developing High-Frequency Trading Systems

02
Mois 3–4 Systèmes
C++ bas niveau & réseau
Maîtriser le C++ moderne orienté performance + sockets UDP/TCP pour les flux de marché
+
Ressources
📘
Effective Modern C++ — Scott Meyers
Move semantics, lambdas, smart pointers, constexpr. Ce sont exactement les features utilisées dans le code de prod HFT en C++17/20.
Payant
📗
The Linux Programming Interface — Kerrisk
Sockets, threads POSIX, gestion mémoire, signaux. La bible du bas niveau Linux — ce que les equipes HFT attendent de toi.
Payant
▶
Coursera — C++ for C Programmers (UC Santa Cruz)
Accessible en audit gratuit. Parfait pour systématiser tes connaissances C++ avant de plonger dans la gestion mémoire avancée.
Audit gratuit
{ }
LeetCode — 2h/semaine, focus C++
Medium/Hard. Patterns prioritaires : queues, hash maps, sliding window, binary search. Les entrevues HFT testent ces structures.
Gratuit
◈
Mini-projet : serveur UDP «exchange simulé»
Écrire un serveur UDP en C++ qui lit un CSV de ticks et les streame en local à 10 000 messages/sec. Pas d'UI — juste des sockets bruts.
Projet

◉ Jalon : Un serveur UDP C++ tourne en local, streame des ticks financiers CSV simulés, et tu peux expliquer chaque ligne du code lors d'un entretien.
