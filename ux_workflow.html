Recherche
Empathy maps Pain point matrix Personas JTBD statements
Comment utiliser : Clique sur un prompt pour le copier dans le presse-papier. Remplace les [crochets] par ton contexte projet. Les badges + indiquent les ajouts recommandés au-delà du workflow UX standard.
🗺️
Empathy map
Pense, ressent, dit, fait — 4 quadrants par segment utilisateur
Prompt IA
Génère une empathy map pour [persona] qui essaie de [tâche]. Inclus 4 quadrants avec citations fictives réalistes. ⎘
⚡
Pain points +
Frustrations fonctionnelles, émotionnelles & sociales — priorisées
Prompt IA
Liste et priorise les pain points de [persona] autour de [domaine]. Catégorise et crée une matrice de sévérité. ⎘
👤
Persona
Profil archétypal basé sur la recherche terrain
Prompt IA
Crée un persona UX complet pour [produit] : démographie, objectifs, frustrations, comportements digitaux, citation clé. ⎘
🎯
Jobs-to-be-Done +
Ce que l'utilisateur cherche vraiment à accomplir (JTBD)
Prompt IA
Formule les JTBD pour [persona] : jobs fonctionnels, émotionnels et sociaux au format Quand/Je veux/Pour que. ⎘
💡
Problem statement
HMW (How Might We) + énoncé du problème centré utilisateur
Prompt IA
Formule 3 HMW statements à partir de [pain points]. Puis rédige 1 problem statement principal centré utilisateur. ⎘
🔬
Hypothesis statement
Si [on fait X] → [utilisateur atteint Y] → [on mesure Z]
Prompt IA
Rédige 3 hypothèses pour [problème] au format "Nous croyons que / Nous saurons que". Classe par niveau de risque. ⎘
💎
Value proposition
Canvas de valeur : gains, pains, créateurs de valeur
Prompt IA
Remplis un Value Proposition Canvas pour [produit] + [persona]. Identifie les points de fit les plus forts. ⎘
⚙️
Cas d'utilisation +
Scénarios fonctionnels acteur-système (hérité génie logiciel)
Prompt IA
Génère 5 cas d'utilisation pour [système] : acteur, préconditions, flux principal et alternatifs, postconditions. ⎘
📋
User stories
En tant que [rôle], je veux [action] pour [bénéfice]
Prompt IA
Génère 10 user stories pour [produit] + [persona]. Format Gherkin pour les 3 critiques. Ajoute critères d'acceptation. ⎘
🗺️
User journey map
Phases, actions, pensées, émotions, opportunités
Prompt IA
Crée une journey map pour [persona] + [tâche] : phases, actions, émotions, frictions, opportunités. ⎘
🎬
Storyboard
Séquence visuelle narratif du contexte d'usage réel
Prompt IA
Rédige un storyboard 6 scènes pour [persona] utilisant [produit] : contexte, action, dialogue, émotion. ⎘
🔷
Service blueprint +
Frontstage, backstage, infrastructure — vue système complète
Prompt IA
Crée un service blueprint pour [service] : frontstage, backstage, ligne de visibilité, systèmes support. ⎘
🔀
User flow
Chemins de navigation entre écrans et états
Prompt IA
Décris le user flow pour [tâche] dans [produit] : écrans, décisions, transitions, états d'erreur. ⎘
🏗️
Architecture IA +
Sitemap, card sorting, taxonomie du contenu
Prompt IA
Propose une architecture d'information + sitemap pour [produit]. Justifie chaque regroupement selon les modèles mentaux. ⎘
🕸️
Modèle de domaine +
Entités métier, attributs, relations (hérité génie logiciel)
Prompt IA
Crée le modèle de domaine pour [produit] : entités, attributs, relations et cardinalités. ⎘
↔️
Diagramme interaction +
Séquences UI ↔ système ↔ API (hérité génie logiciel)
Prompt IA
Décris le diagramme d'interaction pour [feature] : utilisateur → UI → backend → BDD, messages et réponses. ⎘
🎨
Moodboard
Direction artistique, références visuelles, ton & ambiance
Prompt IA
Décris le moodboard pour [produit] + [persona] : ton, palette, typo, références visuelles à inclure et éviter. ⎘
🎨
Color system
Primaires, secondaires, sémantiques, neutrals, dark mode
Prompt IA
Génère un color system complet pour [produit] : primaire, secondaire, sémantiques, neutrals + ratios WCAG. ⎘
📝
Type system
Font pairing, échelle typographique, line-height, tracking
Prompt IA
Propose un type system pour [produit] : font pairing, échelle Display→Caption, line-heights, weights et cas d'usage. ⎘
⚌
Grid & spacing +
Colonnes, gouttières, base-8 spacing, breakpoints
Prompt IA
Définis grille + espacement pour [produit] : colonnes mobile/tablet/desktop, base-8 spacing scale, breakpoints. ⎘
⚛️
Atomes
Boutons, inputs, icônes, badges, avatars, chips
Prompt IA
Liste tous les atomes pour [produit] avec leurs états, variantes et props Figma. Organisés par catégorie. ⎘
🧩
Molécules & organismes
Cards, navbars, forms, modals, tables, headers
Prompt IA
Définis molécules et organismes pour [produit] : composition, variantes, responsive, états. Groupés par fonction. ⎘
🏷️
Design tokens +
Variables Figma exportables : couleurs, espacements, radius, ombres
Prompt IA
Génère les design tokens JSON pour [produit] au format Style Dictionary : couleurs, spacing, typo, radius, shadows, dark mode. ⎘
♿
Accessibilité +
WCAG AA, contrastes, focus visible, ARIA guidelines
Prompt IA
Audite l'accessibilité du design system de [produit] : contrastes WCAG, focus, aria labels, navigation clavier. ⎘
✏️
Lo-Fi wireframes
Grilles, blocs, hiérarchie — sans couleur ni style
Prompt IA
Décris la structure Lo-Fi de [écran] : blocs de contenu priorisés, navigation, actions, patterns UI. ⎘
📱
Hi-Fi mockups
Design final appliqué — couleurs, typo, composants réels
Prompt IA
Décris le passage Lo-Fi→Hi-Fi pour [écran] : composants, hiérarchie visuelle, micro-interactions, adaptation mobile. ⎘
✨
Micro-interactions +
Animations, états de chargement, transitions, feedback
Prompt IA
Propose les micro-interactions pour [composant] : trigger, animation, durée, easing, but fonctionnel et impact émotionnel. ⎘
✍️
UX writing +
Titres, labels, CTAs, erreurs, onboarding copy
Prompt IA
Rédige l'UX writing pour [écran] : headline, labels, CTAs, erreurs, états vides, messages succès. ⎘
🔗
Prototype Figma
Connexions, transitions, overlays, scroll behavior
Prompt IA
Définis le plan de prototypage Figma pour [flux] : connexions, triggers, transitions, overlays, edge cases. ⎘
🧪
Usability testing +
Script de test, tâches, métriques SUS, taux de complétion
Prompt IA
Rédige le script de test utilisateur pour [produit] : 4 scénarios, think-aloud, questionnaire SUS, questions de clôture. ⎘

Itération
